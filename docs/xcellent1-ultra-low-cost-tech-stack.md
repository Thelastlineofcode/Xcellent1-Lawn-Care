# Xcellent1 — Ultra Low-Cost Tech Stack (summary)

This short summary outlines the recommended ultra-low-cost, open-source-friendly
stack for the project. The full detailed cost analysis and scenario breakdown is
archived at `docs/archive/xcellent1-ultra-low-cost-tech-stack.full.md`.

Key points:

- Recommended stack: Next.js (Vercel) front-end, Node.js API, Supabase for DB &
  Auth, Twilio for SMS, SendGrid for email, Stripe for payments,
  LangChain/OpenAI for optional agents.
- Cost profile: Year 1 infrastructure + domain roughly $200–$600; ongoing
  $40–$150/month depending on usage.
- Benefits: Low overhead, agent-friendly, ownable/open-source stack; great for
  small business scale (1–1000 customers).

See archived document for full comparisons, cost tables, and implementation
steps.
