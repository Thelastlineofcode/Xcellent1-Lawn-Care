# South Louisiana Blog WFD Framework

## Goal
Build a South Louisiana lawn-care blog that acts like a Stripe-grade docs system:
- clear scope
- predictable structure
- exhaustive coverage
- strong internal linking
- direct conversion to quote requests and hires

## ERC framing

### Empathize
- People search for simple answers about Louisiana lawns.
- The market is humid, seasonal, flood-prone, and local-intent heavy.
- The blog has to answer real questions fast and build trust.

### Realize
- The site already has:
  - a blog hub at `/blog`
  - a homepage blog section
  - a sitemap entry
  - service-area and quote CTAs
- The current blog only covered a few topics.

### Conceptualize
- Cover the full South Louisiana lawn-care stack with pillar posts.
- Make each post locally specific.
- Tie every post back to the business.

## WFD rules

1. **One article = one search intent**
   - no mixed-topic filler
   - every page should answer a single local query

2. **Stripe-level clarity**
   - short title
   - concrete meta description
   - one clear CTA
   - no fluffy generic advice

3. **Local specificity**
   - mention LaPlace / Reserve / Garyville / River Parishes where relevant
   - include South Louisiana weather, grass types, and seasonal timing

4. **Internal link discipline**
   - every article links back to `/blog`
   - every article links to home, services, quote, and at least 2 related articles

5. **Schema + SEO basics**
   - article JSON-LD
   - canonical URL
   - OG/Twitter tags
   - sitemap entry

6. **Revenue tie-in**
   - each post should end with a quote / service / careers CTA
   - traffic should flow into leads and worker applications
